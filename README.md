# 케어메이트 동행센터 랜딩페이지

병원동행 서비스 홍보와 상담 신청을 위한 정적 랜딩페이지입니다.

## 구성

- `index.html`
- `css/style.css`
- `js/main.js`
- `images/`

## 배포

GitHub Pages에서 `main` 브랜치의 루트 폴더를 배포 대상으로 설정하면 바로 사용할 수 있습니다.

## 상담 신청 폼

현재 상담 신청 폼은 프론트엔드 데모용으로 브라우저 `localStorage`에 저장됩니다. 실제 상담 신청을 받으려면 Google Forms, Google Apps Script, Formspree, Netlify Forms, 별도 API 서버 중 하나와 연결해야 합니다.
